Pegi

An Adobe AIR app that enables connections through media.

Pegi is an app that was built in Adobe AIR. It uses AIR ANE libraries that enable it to scan the barcode of any piece of media and then create a chat-room for that media. If someone were to scan the movie 'Pacific Rim' then they would be loaded into the chat room for that movie; if it didn't already exist then it would be created.

  • Date: May 2018
  • Class: GIMM 350
  • Category: Development

Tangible Design

A design standard for Freeman Co.

At the start of my internship with Freeman Co in June 2018 I was given a style guide called 'Tangible'. It had been developed by the UX / UI Manager as a style guide to standardize the styling all of Freeman Co's web apps. It has been my job thus far to develop this guide in React using Bootstrap naming convention for ease of development. When the project is completed it will be a React component library, a CSS library, and a web page containing all of the design specifications.

  • Date: June 2018 - Present
  • Client: Freeman Co
  • Category: UI / UX Design and Development

Live

A Virtual Reality experience.

Live is a virtual reality experience that I have been working on as a part of my GIMM degree. It consists of a VR game which simulates sitting in the forest around a fire. Combined with that are a series of props including: a space heater, sleeping bag, and fans that are all aligned and synced with objects in game to make the user believe what they are seeing in the VR world is real.

  • Date: Fall 2017
  • Client: GIMM
  • Category: VR
